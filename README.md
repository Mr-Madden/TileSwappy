# TileSwappy

Addictive tile puzzle madness - rotate and swap gradient tiles to match adjacent edges.

🎮 **[Play TileSwappy](www.tileswappy.com)** *(add when deployed)*

## Features
- Daily puzzles with streak tracking
- 7-day puzzle carousel
- Full puzzle archive
- Player statistics

## Tech Stack
- React + TypeScript
- Tailwind CSS
- Supabase (Database + Storage)

## Local Development
```bash
npm install
npm start
```
